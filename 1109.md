- y(0) = 3은 initial point
- y는 t에 대한 함수임. 테일러 expansion
- y'은 t와 y에 대한 함수임.
- higher h는 무시한다. 

```Matlab
clear ; clf ; 
y = @(t) 3*exp(t) + t ; 
t = linspace(0, 4, 100) ; 
plot(t, y(t))
title('Daeun Lee')
```

### Exact solution 찾기 
```Matlab
clear ; clf ; 
y = @(t) 3*exp(t) + t ; 
f = @(t, y) 1-t+y; 

t = linspace(0, 4, 100)      % time (final time = 4) 
plot(t, y(t)); hold on  

T(1) = 0      % time 0에서 시작 
Y(1) = 3      % 여기서 argument는 index (첫번째 값 = 3) 
plot(T(1), Y(1), 'ro')       % exact solution 

h = 0.5 ;     % step size : 작을수록 exact solution이랑 가까워져서 accurate 해짐 
T(2) = T(1) + h;      
Y(2) = Y(1) + h* f(T(1), Y(1));
% plot(T(2), Y(2), 'ro')
plot(T, Y, 'ro-')

title('Daeun Lee')
```

### Generalize 하기 
```Matlab
clear ; clf ; 
y = @(t) 3*exp(t) + t ; 
f = @(t, y) 1-t+y; 

t = linspace(0, 4, 100)      % time 
plot(t, y(t)); hold on  

T(1) = 0      % time 0에서 시작 
Y(1) = 3      % 여기서 argument는 index (첫번째 값 = 3) 
plot(T(1), Y(1), 'ro')       % exact solution 

% Generalize 
Nt = 400 ;       % 많을수록 정확해짐 
h = 4/Nt ; 
for i = 1:Nt
    T(i+1) = T(i) + h ; 
    Y(i+1) = Y(i) + h* f(T(i), Y(i)) ; 
end 
% Y(end) = Y(Nt+1)임 
% [y(4) Y(Nt+1)] : 실제값 / 추정값 

plot(T, Y, 'ro-')
title('Daeun Lee')
```

### Error plotting 
```Matlab
clear ; clf ; format long 
y = @(t) 3*exp(t) + t ; 
f = @(t, y) 1-t+y; 

T(1) = 0      % time 0에서 시작 
Y(1) = 3      % 여기서 argument는 index (첫번째 값 = 3) 

% Generalize 
for k=1:20
    tic               % tic + toc : 실행시간 체크 

Nt = 16 * 2^k ;       % 많을수록 정확해짐 
h = 4/Nt ; 

for i = 1:Nt
T(i+1) = T(i) + h ; 
Y(i+1) = Y(i) + h* f(T(i), Y(i)) ; 
end 

% Error 계산 
err(k) = abs(y(4) - Y(Nt+1)); 
toc 
end 

plot(err, 'ro-') ; grid on 
title('Daeun Lee')
```
